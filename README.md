# YouBike 站點搜尋 Android版

**使用 Jetpack Compose 開發的 YouBike 站點搜尋器**

---

**簡介**

此專案為一個簡單的Android應用程式，提供使用者可以搜尋全國的 YouBike 站點資訊，並查看各站點的車輛狀況。

使用者可以透過輸入站點名稱或定位，快速查詢 YouBike 2.0 站點的資訊，包括：
* 站點名稱
* 地址
* YouBike 2.0 數量
* YouBike 2.0E 數量
* 可停空位數

**其他功能**
* 黑暗模式
* 跟隨系統主題
* 自動刷新
